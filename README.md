# 🛍️ Omzi Shopping Website

A modern and responsive e-commerce website built using HTML, CSS, and JavaScript.

## ✨ Features

* 🛒 Add to Cart functionality
* 🔍 Search products
* 💸 Discount pricing with original price
* 📱 Mobile responsive design
* ❤️ Wishlist (optional feature)
* 🖼️ Product images and categories

## 📂 Project Structure

* index.html (Home page)
* shop.html (Product listing)
* cart.html (Cart page)
* css/ (Styles)
* js/ (Scripts)
* images/ (Product images)

## 🚀 How to Use

1. Open the website link
2. Browse products
3. Add items to cart
4. View cart and checkout

## 💡 Technologies Used

* HTML
* CSS
* JavaScript
